# Volumetric SDK

The Volumetric SDK enables you to bring your existing Windows apps and content into Mixed Reality. You can continue using your familiar workflows and development tools on Windows to extend Win32 apps into Mixed Reality with Volumes.

For more details about the Volumetric SDK, see the documentation website <http://microsoft.github.io/volumetric-sdk-docs>

This repository contains the Volumetric SDK source code, including the libraries and sample code in three programming languages:

* [Develop Volumetric app in C#](https://microsoft.github.io/volumetric-sdk-docs/develop/csharp/)
* [Develop Volumetric app in C++](https://microsoft.github.io/volumetric-sdk-docs/develop/cpp/)
* [Develop Volumetric app in Python](https://microsoft.github.io/volumetric-sdk-docs/develop/python/)

## Repository Structure

```
├── assets/           # GLTF assets for sample apps
├── cpp/include       # C++ Volumetric library (header only)
├── cpp/samples       # Sample C++ Volumetric apps
├── cs/Library        # C# Volumetric library
├── cs/Samples        # Sample C# Volumetric apps
├── python/package    # Python Volumetric package
├── python/samples    # Sample Python Volumetric apps
├── tools/            # Tools for developing Volumetric apps
├── ...
```
